# PWA 安裝指南

## 📱 什麼是 PWA？

PWA (Progressive Web App) 讓您的網頁應用可以像原生 App 一樣安裝到手機主屏幕，無需經過 App Store 審核！

## ✨ PWA 優勢

- ✅ **可安裝到主屏幕** - 像原生 App 一樣有獨立圖標
- ✅ **全屏體驗** - 無瀏覽器地址欄，更專業
- ✅ **離線支持** - 網絡不穩定時仍可查看已緩存內容
- ✅ **啟動更快** - 緩存機制讓啟動速度提升
- ✅ **無需審核** - 直接部署，更新即時生效

## 🚀 快速開始

### 1. 生成圖標

1. 打開 `icon-generator.html` 文件
2. 自定義圖標樣式（顏色、文字、大小）
3. 點擊下載按鈕，下載兩個圖標文件：
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
4. 將圖標文件放到與 `index.html` 相同的目錄

### 2. 部署到 Git

將以下文件上傳到您的 Git 倉庫：

```
InventorySystem/
├── index.html
├── manifest.json          ← PWA 配置文件
├── service-worker.js      ← 離線緩存服務
├── icon-192.png          ← 圖標（需要生成）
└── icon-512.png          ← 圖標（需要生成）
```

### 3. 在手機上安裝

#### iOS (Safari):
1. 打開您的 Git 網址
2. 點擊底部的「分享」按鈕
3. 選擇「加入主畫面」
4. 點擊「加入」

#### Android (Chrome):
1. 打開您的 Git 網址
2. 瀏覽器會自動顯示「安裝應用程式」橫幅
3. 點擊「安裝」或「新增至主畫面」

#### 其他瀏覽器:
- **Chrome/Edge**: 地址欄右側會顯示「安裝」圖標
- **Firefox**: 菜單 → 「安裝網站為應用程式」

## 📋 文件說明

### manifest.json
PWA 的配置文件，定義了：
- 應用名稱和描述
- 圖標路徑
- 顯示模式（全屏）
- 快捷方式（從主屏幕長按圖標）

### service-worker.js
Service Worker 負責：
- 緩存頁面資源
- 提供離線支持
- 自動更新檢測

### icon-generator.html
圖標生成工具，可以：
- 自定義圖標樣式
- 預覽不同尺寸
- 一鍵下載所需圖標

## 🔧 自定義設定

### 修改應用名稱
編輯 `manifest.json`：
```json
{
  "name": "您的應用名稱",
  "short_name": "簡短名稱"
}
```

### 修改主題顏色
編輯 `manifest.json`：
```json
{
  "theme_color": "#000000",
  "background_color": "#000000"
}
```

### 添加快捷方式
在 `manifest.json` 的 `shortcuts` 數組中添加新項目。

## 🐛 常見問題

### Q: 為什麼沒有看到「安裝」提示？
A: 確保：
1. 使用 HTTPS（GitHub Pages 自動提供）
2. 已創建 `manifest.json` 和圖標文件
3. Service Worker 已註冊成功

### Q: 圖標顯示不正確？
A: 檢查：
1. 圖標文件路徑是否正確
2. 圖標尺寸是否為 192x192 和 512x512
3. 清除瀏覽器緩存後重試

### Q: 如何更新應用？
A: 
1. 修改代碼後推送到 Git
2. Service Worker 會自動檢測更新
3. 用戶下次打開時會提示更新

### Q: 可以離線使用嗎？
A: 
- 頁面結構會緩存，可以離線查看
- API 請求需要網絡連接（即時數據）
- 已查看過的頁面可以離線訪問
